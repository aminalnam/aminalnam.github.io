Layer 1: Sensing & Mapping Engine

How OMEGA Works.

OMEGA combines field logging, edge point validation, and map generation in one workflow. The Arduino Mega records GPS, sonar, water temperature, IMU orientation, and UTC time, filtering raw inputs into a mapping dataset.

Overview

Organized as a measurement pipeline.

The system moves from acquisition to validation, from validation to logging, and from logging to reconstruction. Each stage has a different job, and each stage leaves a record that can be inspected later.

Workflow

From field reading to map output

1. CaptureGPS, sonar, temperature, IMU, and UTC time are sampled.
2. ValidateFix quality, depth stability, motion, and spacing are checked.
3. LogThe logger writes the complete record and a clean map dataset.
4. ReconstructPython builds depth surfaces, contours, and overlays.
Codebase

Where the logic lives

  • rov_logger_mapping.ino handles sensor input, validation, display, and SD logging.
  • csv_to_png_depth_interpolated.py builds an IDW depth surface from survey points.
  • csv_to_png_depth_contours.py generates contour plots.
  • csv_to_kmz_depth_overlay.py exports georeferenced Google Earth overlays.
System structure

Sensing, Validation, and Mapping.

Sonar estimates distance from travel time. GPS attaches location and UTC time. Temperature provides sound-speed correction. Validation determines if a point is kept. Python scripts reconstruct a continuous surface.

Thresholds

Explicit Mapping Conditions.

The Arduino sets limits for GPS quality, fix age, platform motion, and sonar correction. Those values determine whether a point is kept.

ParameterCurrent valueRole in the survey
sonarCalSoundSpeed1500.0 m/sBaseline sound speed used when correcting sonar range.
sonarCalTempC20.0 °CReference temperature for the baseline sound-speed value.
maxMapHdop1.8Rejects fixes with poorer horizontal precision.
maxMapFixAgeMs2000 msRejects stale fixes.
maxMapSpeedKmph8.0 km/hLimits mapping when moving too quickly.
maxMapPitchDeg10.0°Rejects points when pitch exceeds allowed range.
maxMapRollDeg10.0°Rejects points when roll exceeds allowed range.
