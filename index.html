Featured project

OMEGA

OMEGA is a compact underwater survey system that records depth, position, and environmental data from a moving ROV, then transforms those measurements into mapped outputs. The public repository includes the logger code, processing scripts, documentation, example data, and generated visual products.

Oceanic Measurement & Environmental Geospatial Array

From field acquisition to mapped surface.

The project combines a hardware capture layer with a processing workflow designed to turn irregular field measurements into usable maps. The system emphasizes stable logging, readable structure, and outputs that can be interpreted quickly in both research and presentation contexts.

Capture GPS, depth, temperature, orientation, onboard display, SD logging
Processing Filtering, interpolation, contours, overlays, KML export
Outputs Raster maps, contour maps, Google Earth overlays, filtered data products
Public material Repository, Hackaday project page, documentation, example files

Working method

Capture, process, publish.

The technical work presented here moves between hardware, software, and presentation. The point is not only to build a system, but to make the system legible once it exists.

01

Capture

Build or adapt a sensing platform, define logging rules, and collect field data that is structured and useful.

02

Process

Filter and interpret raw measurements, then transform them into maps, overlays, documents, and other readable outputs.

03

Publish

Present the work clearly enough that visitors understand what the system does, how it works, and where to go next.
