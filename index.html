OMEGA.

The Oceanic Measurement and Environmental Geospatial Array (OMEGA) is a mixed-bearer environmental sensing network. OMEGA connects field devices, stationary sensor nodes, relays, and gateways into one resilient observation network using the Custom Binary TLV Protocol over LoRa.

Core Network

OMEGA Platform

A massive, mixed-bearer environmental sensing network. Utilizing custom Delay Tolerant Networking (DTN) over Sub-GHz LoRa to democratize oceanographic data collection and replace expensive satellite uplinks.

Testing Engine

Proof Lab

A standalone deterministic evaluation engine. Proof Lab continuously subjects backend gateways to high-frequency synthetic traffic to detect regressions. It is currently configured to relentlessly test the OMEGA network, providing empirical proof of its resilience.
