SYSTEM.01 Live Repository

Oceanic Measurement & Environmental Geospatial Array

A compact underwater survey system that collects depth, position, and environmental data from a moving ROV and converts those measurements into bathymetric maps and georeferenced outputs.

  • Arduino-based capture device with onboard logging
  • Python tools for maps, contours, interpolation, and overlays
  • Field-to-visualization workflow for environmental measurement

What the repository includes

  • ROV logger code and hardware-facing capture logic
  • Example datasets for testing the mapping pipeline
  • Scripts for KML export, Google Earth overlays, and depth maps
