Embedded UI & Autonomous Agents

Visual Timer.

A configurable, glanceable e‑ink timer that tells time as 101 tiny animated story scenes.

Art Portfolio
At A Glance

Scaling Embedded Content

The Visual Timer runs on the Elecrow CrowPanel ESP32‑S3 2.13" E‑Paper display. But rather than hand-coding firmware, this project is a showcase of building a production pipeline for autonomous design agents to generate 101 distinct, stateless C++ animations—audited and integrated by Python scripts.

Visuals

101 Animated Stories

Every face is a pure stateless function that draws a scene—like a rising tide, an eclipsed moon, or a Rube Goldberg machine. Hand-coding these would take years; an LLM agent fleet did it systematically.

Pipeline

1-Bit Render Mirrors

A Python/Pillow shim perfectly re-implements the ESP32 GFX primitives on the desktop. The LLM agents write C++, render a verification strip in Python, and visually critique their own output against a contract.

Quality Gate

Automated Fleet Audit

A fleet of 101 independent agents reads the visual verification strips and grades them on dynamism, legibility, and narrative escalation, producing a ranked fix list without human intervention.

Agentic Workflows

Deterministic Integration.

Giving a generative AI agent the ability to see its own output changes everything. The agents render a 3-row verification strip: Row A is the scene swept across the countdown in landscape; Row B in portrait; and Row C tests real-time motion at fixed progress.

Once an agent's scene passes its self-critique, a set of deterministic Python scripts seamlessly splices the verified C++ code directly into the firmware. The scripts validate brace balances, forbid standard-library imports, collapse dispatchers, and generate enums, ensuring the final output compiles identically every single time.

Power Engineering

The E-Ink Lifecycle

Graceful Sleep

Deep-Sleep Cycling

When paused or finished, the ESP32-S3 enters deep-sleep mode, drawing only tens of microamps. The bistable display controller retains the final image frame infinitely with zero power.

Hardware Quirks

Driver Profiling

Identical part numbers don't guarantee identical glass. On a replacement board, the SSD1680 panel ghosted severely. Diagnosing the issue required switching the GxEPD2 controller profile (to GDEY0213B74), proving the necessity of confirming driver waveforms against actual physical batches.
