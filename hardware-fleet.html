Hardware Nodes

The Fleet

Mobile gateways, shore controllers, and deployable relay structures making up the physical layer of the OMEGA mesh network.

ASV Mobile Gateway

The first deployable node profile acts as an Autonomous Surface Vehicle (ASV) and mobile gateway. It utilizes a Pixhawk Jetson Baseboard Bundle with a Jetson Orin NX for local autonomy and edge-compute tasks (sensor fusion, bathymetry, compression).

Mixed-Bearer Hardware

The ASV carries multiple radios for redundancy: LilyGO T-HaLow for IP backhaul, LilyGO T-Beam Supreme for GNSS/IMU/Telemetry, and LilyGO T-Beam 1W for strong sub-GHz LoRa transmission, plus LTE support.

Disposable LoRa Relay Chains

When the direct shore link loses margin, the ASV can drop cheap floating LoRa relays to preserve the control chain. The planner calculates desired spacing, hop targets, and RSSI triggers to create a path like `shore -> relay-01 -> ASV`.

Shore Controller

Handles the shore-to-ASV bring-up plan, providing the gateway API, shore HaLow AutoInterface, and LoRa CBOR bridge. The safety boundary ensures OMEGA-wave only sends mission intents, while Pixhawk retains manual override control.
