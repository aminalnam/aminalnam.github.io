Fluid Dynamics & Vector Math

Course Over Ground (COG).

An ASV operates on the surface of a moving fluid. If it points its bow directly at a target (its Heading), but a strong cross-current pushes against the hull, its actual path over the Earth (its Course-Over-Ground, or COG) will drift significantly off-target.

The Wayfinder module models the vessel's kinematics as one vector, and imports real-time ocean current data as an environmental vector field to calculate the necessary "crab angle".

Algorithm Design

A* Routing on Dynamic Vectors.

Standard terrestrial routing algorithms assume static roads. Marine routing requires navigating a dynamic fluid field. OMEGA implements adapted A* algorithms that heavily penalize nodes featuring counter-currents, finding paths that optimize for fuel efficiency rather than strict linear distance.

Wayfinding Execution

Kinematic Route Generation.

This is a raw wayfinding-plan.json payload. An operator submits this structure to the Gateway. The backend pathing engine runs multiple parallel fluid dynamic simulations using the speeds_knots and route_offsets_nm arrays, scoring each variation against live wind and wave polygons extracted from the OMEGA database.

wayfinding-plan.example.json
{
  "origin": {
    "lat": 36.7998,
    "lon": -121.7902,
    "label": "Harbor A"
  },
  "destination": {
    "lat": 36.806,
    "lon": -121.7808,
    "label": "Survey Line 1"
  },
  "objective": "balanced",
  "routing": {
    "speeds_knots": [3, 5, 7],
    "route_offsets_nm": [-2.5, 0, 2.5],
    "min_water_depth_m": 12.0
  },
  "environment": {
    "site": "wayfinding-a",
    "stale_after_s": 21600,
    "influence_radius_nm": 20.0
  }
}
Coordinate Autonomy

ASV Waypoint Missions.

Autonomous Surface Vehicles (ASVs) can execute strict coordinate waypoint sequences. The Mission Portal builds JSON payloads defining bounding geofences, speeds (in meters per second), and camera polling rates. This allows for complex survey grids during operations.

Hydrodynamics

HFRNet Surface Currents.

Navigating an ASV against a 4-knot current drains the battery instantly. The hfrnet.py module ingests the High-Frequency Radar Network data, providing real-time vector fields of ocean surface currents. The wayfinding.py module consumes this vector matrix, executing A* pathfinding that "surfs" the currents, drastically extending operational range.

Autonomous Safety

Active Guard Geofencing.

The active_guard.py module is a merciless watchdog. It constantly calculates the cross-track error of every ASV against predefined GeoJSON exclusion zones. If a vessel deviates into a shipping lane due to high winds, the Active Guard instantly overrides the standard waypoint mission and executes an emergency heading correction.
