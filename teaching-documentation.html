The Journey

From the Ocean to the GPU

To truly understand OMEGA, you must trace the flow of data. A system is not just code; it is a pipeline. Here is the exact path a single water temperature reading takes through the entire stack.

sequenceDiagram participant O as Ocean (Physical) participant E as ESP32 Edge Node (C++) participant R as LoRa Relay (Airwaves) participant G as Python Gateway (ASGI) participant D as SQLite WAL (Disk) participant W as React Portal (WebGL) O->>E: Analog Thermistor (12-bit ADC) E->>E: Compress into Binary TLV E->>R: 915MHz RF Transmission (LoRa CSS) R->>G: USB CDC Serial UART Bridge G->>G: Unpack Binary to JSON G->>D: Async SQL INSERT (WAL mode) D-->>G: Acknowledge Write G->>W: WebSocket Push (10Hz) W->>W: deck.gl GPU Context Render
Phase 1: The Benthic Layer

Analog to Digital

The journey begins in the physical world. The ocean water cools the DS18B20 thermistor probe. This physical temperature drop increases the electrical resistance of the wire. The ESP32's internal Analog-to-Digital Converter (ADC) reads this voltage drop and maps it to an integer (e.g., `2048` = `12.5°C`).

Phase 2: The Edge Compute

Binary TLV Packing

The C++ firmware (`src/main.cpp`) intercepts this integer. Sending `{ "temperature": 12.5 }` over the radio wastes precious bandwidth. Instead, the code packs it into a raw Type-Length-Value (TLV) byte array. The reading is crushed down into exactly 3 bytes: `[0x01, 0x01, 0x7D]`.

Phase 3: The Transport Layer

Chirp Spread Spectrum

The C++ array is passed to the SX1262 LoRa transceiver. The silicon modulates the 3 bytes into a sweeping analog radio frequency (Chirp Spread Spectrum) and blasts it out the antenna at 1-Watt. The invisible RF wave bounces across the ocean surface, fighting through fog and saltwater absorption, until it physically strikes the antenna of the Shore Gateway.

Phase 4: The Shore Gateway

Python & ASGI

The Shore Relay catches the wave, converts it back to the 3 bytes, and pushes it over a USB cable. The Python `asyncio` event loop catches the bytes via `pyserial`. The Python code unpacks the TLV back into a Python Dictionary: `{"temp": 12.5}`. It then immediately triggers a non-blocking `INSERT` query into the SQLite `.db` file using WAL mode to prevent SD card locking.

Phase 5: The Presentation Layer

React & GPU WebGL

Simultaneously, the Python FastAPI router fires the JSON over an active WebSocket connection. A React frontend on a scientist's laptop receives the payload. Instead of manipulating standard HTML, React passes the coordinate and temperature data directly into `deck.gl`, writing the data directly to the computer's Graphics Card (GPU) memory. A glowing red point instantly appears on the 3D map.

From the freezing ocean thermistor to the glowing pixel on a screen in under 400 milliseconds.
