Data Intelligence

Environmental Context

Ingesting, normalizing, and structuring both local field sensor readings and credentialed global datasets into a unified spatial model.

Observation Normalization

The gateway ingests canonical observation envelopes over HTTP and normalizes measurements into an SQLite backend. This exposes an online-friendly REST/JSON and OpenAPI interface, allowing simple querying without worrying about the radio specifics.

Public Ocean & Weather

OMEGA actively pulls in public context from NOAA NDBC buoys, NOAA CO-OPS tide products, Open-Meteo marine models, and NWS weather alerts. This allows users to combine live sensor readings with massive public forecast structures.

Global Fishing Watch (GFW)

A credential-safe provider pipeline seamlessly injects the 4Wings API for real heatmap overlays, apparent fishing effort, vessel presence (AIS), and satellite SAR detections into the map portal—without exposing your tokens to the client browser.

Commercial Data Providers

Direct integration for platforms like Stormglass (marine weather), Copernicus Marine, NASA EarthData, and AISStream, bridging commercial and research-grade data down into the localized operational mesh map.
