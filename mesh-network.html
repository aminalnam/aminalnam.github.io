Mesh Core

Robust Data Routing

A first-pass mesh control plane managing interfaces, neighbors, and packet histories across diverse bearers including TCP, LoRa, and Meshtastic.

Mesh Interface Registry

Maintains interface presence, neighbor tracking, and gateway-boundary duplicate packet protection. Features a store-and-forward outbox architecture that handles retries and prevents infinite loops on failed links.

LoRa CBOR Bridge

The primary sub-GHz data bearer utilizing a compact lora-cbor-v1 codec over T-Beam hardware. Supports framed compact CBOR and CRC validation on serial frames, ideal for telemetry and route beacons.

TCP JSONL Bridge

A deterministic lab and backhaul link running TCP JSONL. Coupled with a Reticulum-inspired AutoInterface daemon, it allows zero-config local IP mesh segment setup.

Meshtastic Piggybacking

OMEGA doesn't replace Meshtastic—it leverages it. The python-based meshtastic bridge routes inbound packets into the OMEGA control plane and forwards outbox items through the Meshtastic Python API.
