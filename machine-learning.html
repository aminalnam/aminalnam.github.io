Machine Learning.

Predictive Analytics

Time-Series Forecasting

Plotting current temperatures on a map is useful for monitoring, but it is fundamentally reactive. To truly protect an ecosystem, we must become predictive.

ARIMA Models

Statistical Baselines

For highly seasonal data (e.g., daily tidal heating cycles), we utilize AutoRegressive Integrated Moving Average (ARIMA) models. These statistical models run natively on the Python backend, analyzing the SQLite database to establish the "normal" sinusoidal rhythm of the bay, instantly flagging any statistical deviation.

LSTM Neural Nets

Deep Learning

For complex, multi-variate forecasting (e.g., predicting how wind speed, salinity, and temperature will interact 48 hours from now), we export the data to train Long Short-Term Memory (LSTM) neural networks using TensorFlow. This allows the system to predict coastal thermal anomalies and warn researchers days before coral bleaching actually occurs.

machine_learning/train_lstm.py 
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense, Dropout

def build_thermal_anomaly_model(input_shape):
    model = Sequential([
        # 1. Recurrent Layer for temporal memory (e.g. 48 hours of past data)
        LSTM(64, return_sequences=True, input_shape=input_shape),
        Dropout(0.2), # Prevent overfitting on noisy ocean data
        
        # 2. Deep abstraction layer
        LSTM(32, return_sequences=False),
        Dropout(0.2),
        
        # 3. Output Layer (Predicting T+1 Temperature)
        Dense(16, activation='relu'),
        Dense(1, activation='linear')
    ])
    
    model.compile(optimizer='adam', loss='mse', metrics=['mae'])
    return model
