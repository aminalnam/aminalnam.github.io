Marine Biology

Biofouling.

The unavoidable biological reality of the ocean, and how we mitigate calcification on delicate sensors.

View OMEGA on GitHub Art Portfolio
The Threat

Nature Reclaims Everything

Within hours of dropping a buoy into the ocean, a biofilm of bacteria coats the hull. Within weeks, macro-fouling begins: algae, barnacles, and tube worms attach and begin calcifying.

For a dumb plastic hull, this simply adds weight. But for a delicate acoustic hydrophone or a salinity conductivity probe, biofouling is a death sentence. A thick layer of barnacles will physically muffle an acoustic transducer, destroying the JANUS communication link. Algae growing inside a salinity probe will permanently skew the electrical conductivity readings.

Mitigation Strategies

Chemical & Mechanical Defense

Chemical

Copper Anti-Fouling

The lower portion of the OMEGA hull is painted with ablative Cuprous Oxide (copper) marine paint. As the paint slowly dissolves in the saltwater, the copper ions create a highly toxic micro-environment directly on the hull's surface, preventing barnacle larvae from securing a strong adhesive bond.

Acoustic

FFT Shift Detection

We can mathematically detect barnacles before they destroy the acoustic link. As calcification builds up on the titanium transducer face, the resonant frequency of the hydrophone physically shifts. The Gateway's Fast Fourier Transform (FFT) pipeline detects this micro-shift in the spectrum and flags the node for preventative maintenance.

gateway/services/acoustics.py 
import numpy as np
from scipy.fft import fft, fftfreq

def detect_calcification(raw_audio_buffer: bytes, sample_rate: int = 44100):
    """
    Detects barnacle growth by measuring the downward shift in the
    JANUS transducer's natural resonant frequency (nominal 11.5 kHz).
    """
    # 1. Convert PCM bytes to float array
    audio_data = np.frombuffer(raw_audio_buffer, dtype=np.int16)
    
    # 2. Compute Fast Fourier Transform
    N = len(audio_data)
    yf = fft(audio_data)
    xf = fftfreq(N, 1 / sample_rate)
    
    # 3. Find the peak resonance
    peak_idx = np.argmax(np.abs(yf[:N//2]))
    peak_hz = xf[peak_idx]
    
    # 4. As mass increases (barnacles), resonant frequency decreases
    if peak_hz < 10500: # Dropped from 11.5 kHz to below 10.5 kHz
        return {"status": "FOULING_CRITICAL", "hz": peak_hz}
    return {"status": "HEALTHY", "hz": peak_hz}
