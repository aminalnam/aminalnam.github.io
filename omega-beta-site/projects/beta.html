How Progress Is Judged

  • Comparable baselineThe latest run is compared with the previous run that has the same scenario id, duration, and step size.
  • Direction mattersTest pass rate, render success, project-control coverage, todo progress, and completed evidence should rise. Open blockers should fall.
  • Small noise bandChanges under 2 percent are treated as unchanged so tiny run-to-run noise does not look like progress.
  • Findings become workFailed gates, large payloads, high storage cost, and material regressions become backlog items.
