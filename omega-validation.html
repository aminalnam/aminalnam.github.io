Part 2: Logic

Edge Validation Algorithms.

A mathematically perfect depth measurement is utterly useless if its geographic coordinate is wrong. This chapter breaks down the C++ state machine used to violently reject compromised data before it reaches the mesh network.

The Heuristic Engine

Deciding if a point is acceptable.

In bathymetry, the integrity of the spatial anchor (Lat/Lon) and the orientation of the sensor (Pitch/Roll) are just as critical as the depth reading itself. OMEGA executes a series of strict heuristic gates to validate these parameters. If any gate fails, the data point is aborted.

C++ State Machine

The getMapStatus() Function

const char* getMapStatus(unsigned long currentMillis) {
  // 1. Hardware State Checks
  if (millis() - startupMillis < gpsWarmupTime) return "WARMUP";
  if (!gpsDataPresent(currentMillis)) return "NOGPS";
  
  // 2. Spatial Anchor Checks
  if (!gps.location.isValid()) return "NOFIX";
  if (gps.location.age() > maxMapFixAgeMs) return "STALE";
  if (!gps.satellites.isValid() || gps.satellites.value() < minMapSats) return "SAT";
  
  // 3. Precision Checks (HDOP)
  if (!gps.hdop.isValid() || gps.hdop.hdop() > maxMapHdop) return "HDOP";
  
  // 4. Orientation Checks (IMU Pitch/Roll)
  if (requireLevelForMapping) {
    if (!imuReady) return "NOIMU";
    if (fabs(imuPitchDeg) > maxMapPitchDeg || fabs(imuRollDeg) > maxMapRollDeg) return "UNLVL";
  }
  
  // All gates passed. The data is cleared for transmission.
  return "ARMED";
}
Deep Dive

Why each check exists.

The error states returned by the algorithm correspond to specific geometric or physical distortions that ruin topographical map generation.

Spatial Blur

The HDOP Threshold

Horizontal Dilution of Precision (HDOP) measures the geometric quality of the satellite constellation. An HDOP of 1.0 is ideal. If HDOP spikes to 3.0 (e.g., due to atmospheric interference or multipath reflection near a cliff), the GPS fix might wander by 10 meters.

If we map a 50-meter depth reading to a coordinate that is 10 meters off, we create a massive "spike" or "hole" in the final 3D surface model. OMEGA enforces a strict `maxMapHdop = 1.8`.

Beam Geometry

IMU Pitch and Roll (UNLVL)

Transducers fire a focused cone of sound straight down (the nadir). If the rover pitches violently over a wave, the sonar beam strikes the seafloor at an angle. This measures the hypotenuse of the triangle, not the true vertical depth.

OMEGA's onboard IMU tracks pitch and roll in real-time. If the vessel tilts past `12.0°`, the state machine triggers the `UNLVL` flag, instantly rejecting the ping until the vessel stabilizes.
