OMEGA Architecture

Digital Twin Portal

The interactive "Glass Bridge". A React-based Single Page Application (SPA) providing mission control, live bathymetric surfacing, and predictive wayfinding.

React & Zustand Architecture

The OMEGA-wave portal abandons traditional, clunky mapping frameworks in favor of a hyper-responsive React architecture. By using Zustand for state management, the portal synchronizes its internal simulationTime and fleetState across dozens of UI components instantaneously.

When the FastAPI backend pushes a new WebSocket frame containing a rover's location, the state updates globally. The central MapLayer component, the left-hand diagnostic panel, and the predictive routing lab all re-render simultaneously to reflect the new data.

Live Bathymetric Surfacing

As the ASV rovers traverse the water, their discrete depth soundings appear as distinct points on the map. However, the portal doesn't just show dots—it actively interpolates the space between them.

Dynamic Interpolation

The frontend pulls the latest bathymetry-depth metrics attached to the GPS trail. It runs a lightweight interpolation algorithm in the browser (or requests a fast raster from the backend) to generate a smooth, color-graded bathymetric layer overlaid atop standard satellite basemaps.

Real-Time Control

The portal isn't just for viewing; it controls the mesh. Using the Engineering Deck, operators can queue JANUS acoustic commands or issue LoRa parameter changes to individual nodes directly from the browser.

Predictive Wayfinding Code

Modeled after commercial voyage optimization tools, the wayfinding layer allows operators to simulate survey routes before dispatching the ASV. The tool compares the planned route against live environmental constraints (currents, wind, static obstacle polygons).
