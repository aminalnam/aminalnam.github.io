OMEGA Architecture

Digital Twin Portal

The web-based command center for OMEGA. A React SPA that provides real-time visualization of the autonomous fleet, dynamic mission planning, and live mapping.

Real-Time Fleet Maps

Because the API backend streams data over WebSockets, the portal updates instantly. The "Glass Bridge" map plots the exact geospatial location of every active rover and relay in the mesh.

Live Bathymetry

As rovers capture depth readings, the portal actively traces their paths, color-coding UI elements based on real-time depth, temperature thermoclines, or LoRa signal strength.

Predictive Routing

A simulation lab allows operators to draw predictive survey routes. By analyzing historical current data and node battery life, the UI can simulate optimal pathing before executing commands.

Engineering Deck

A dedicated administrative panel provides deep insights into the mesh network's topology, displaying live packet loss metrics, CBOR compression ratios, and individual node statuses.
