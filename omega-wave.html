OMEGA-wave

The Map Portal

A lightweight spatial portal inspired by top environmental sensor maps. Designed for operational awareness, data visualization, and mesh routing control.

Interactive Mapping

The core of the portal is a map-first interface for exploring stations and fleets.

  • Switchable basemaps (street, topo, satellite).
  • Color-by-metric and freshness-aware styling.
  • Density-grid aggregations for quick regional pattern scanning.
  • NOAA chart and research-overlay selectors.
  • Direct CSV data downloads directly from the portal view.

Bathymetry & Sub-surface

OMEGA-wave can construct first-pass bathymetric maps dynamically from standard observation envelopes with depth soundings.

  • Ingests `bathymetry-depth` metrics attached to GPS coordinates.
  • Generates an interpolated grid map layer.
  • Exports raw sounding point queries to CSV.

Wayfinding & Routing

Modeled after voyage optimization flows, the wayfinding layer allows robust path planning.

  • Scenario comparison (offsets, speeds, departure windows).
  • Scores scenarios against the latest environmental observation data (currents, waves, wind).
  • Land-aware routing utilizing uploaded obstacle polygons.
  • Planned vs. actual track analysis.

Control Center & Diagnostics

The portal isn't just for viewing—it controls the mesh.

  • Live parameter changes and supervisory command dispatch.
  • Mesh operations panels monitoring interfaces, delivery queues, and packets.
  • JANUS test composer and acoustic command queueing tools.
  • Built-in visualization studio (histograms, diagnostics, statistics).
