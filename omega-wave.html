OMEGA-Wave.

OMEGA-Wave is a developing companion project related to communications, distributed sensing, and the movement of field data between devices operating across shoreline and near-water environments.

It does not yet have the same level of public documentation as OMEGA, but this page creates a stable place for the project as it develops into a more distinct system.
