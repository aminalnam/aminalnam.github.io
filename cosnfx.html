Wear OS & IMU Telemetry

CosNFX.

A Wear OS IMU-driven kinetic synthesizer and tactical heads-up display.

Art Portfolio
At A Glance

Kinetic Wearable Platform

CosNFX is an Android Wear OS application that transforms a smartwatch into a kinetic sound and visual effects engine. It leverages high-frequency IMU sensors to translate physical arm movements into dynamic sounds, haptic feedback, and on-screen tactical visuals.

Sensors

8 Recognized Gestures

The underlying HeroSynth engine continuously processes accelerometer and gyroscope telemetry (Pitch, Roll, Accel) to detect specific gestures like punches, uppercuts, shield raises, and whip snaps.

UI Engine

Wear OS Jetpack Compose

The tactical interface is built entirely with Jetpack Compose for Wear OS, featuring a "7-Node Radial Tactical Array" and a central "Identity Reactor Core" that responds fluidly to the user's kinetic states.

Connectivity

Embedded Web Server

The watch runs a local PowersWebServer, enabling it to bridge over local WiFi to a mobile phone, allowing users to select from 20 different VFX profiles remotely.

Kinetic Detection Engine

The HeroSynth Architecture.

Processing raw IMU data in real time on a constrained wearable device requires aggressive optimization to prevent battery drain while maintaining instant responsiveness. The HeroSynth class polls the hardware at high sampling rates, running a localized physics model that applies smoothing functions to the Pitch, Roll, and Acceleration vectors.

When a physical threshold is breached (e.g., a rapid forward acceleration followed by a deceleration and roll twist), the engine fires a trigger event. This translates the abstract math of the sensors into tangible actions like snap_over, punch, or shield_raise.

Network Capabilities

Distributed Combat Links

Health Deductions

Multiplayer UDP Sync

The embedded web server supports a distributed combat mode. It listens for COMBAT_ATTACK network events. When struck virtually by another paired system, the smartwatch instantly calculates a health deduction, vibrating the wrist heavily with a haptic impact_damage routine.

Dynamic Shielding

Kinetic Blocking

If the wearer physically raises their arm into the shield detection posture precisely as a network attack arrives, the HeroSynth blocks the damage, triggering an audio-visual deflection sequence instead.

Diagnostics

The Sensor Lab.

Deploying gesture-based algorithms in the real world is chaotic. To debug the physics model live, the watch features an on-board Sensor Lab. It visualizes the raw incoming telemetry streams directly on the tiny 1.4" display, allowing the developer to tune the threshold bounds of the kinetic simulator while physically wearing the hardware.
