Step 0: The Bill of Materials

Component Sourcing

OMEGA strictly adheres to COTS (Commercial Off-The-Shelf) hardware to eliminate proprietary vendor lock-in. The entire edge node can be sourced for under $100.

6V 2W Solar Panel Trickle Charging (MPPT) $15.00

3D Printing Parameters (Hull Fabrication)

To survive ocean pressure and UV degradation, PLA is completely unacceptable. The hull must be fabricated using PETG or ASA filament. Print parameters must be strictly tuned for structural watertightness rather than speed:

Nozzle: 260°C | Bed: 80°C | Walls: 6 perimeters | Infill: 40% Gyroid | Cooling: 20% max

Step 2: The Firmware

The C++ "Hello World"

Once physically wired, you must flash the C++ firmware using PlatformIO. This code block shows the exact, absolute minimum logic required to initialize the SX1262 silicon over the SPI bus and blast a telemetry packet over the ocean.

src/main.cpp 
#include <Arduino.h>
#include <RadioLib.h>

// Define SPI pins mapping the ESP32 to the Semtech SX1262
#define RADIO_NSS   7
#define RADIO_DIO1  36
#define RADIO_RESET 12
#define RADIO_BUSY  13

// Initialize the LoRa Radio Module
SX1262 radio = new Module(RADIO_NSS, RADIO_DIO1, RADIO_RESET, RADIO_BUSY);

void setup() {
  Serial.begin(115200);
  Serial.println("OMEGA Node Booting...");

  // Initialize Radio: 915 MHz, Bandwidth 125kHz, Spreading Factor 10, Coding Rate 4/5
  int state = radio.begin(915.0, 125.0, 10, 5, 0x34, 17, 8, 1.6);
  if (state == RADIOLIB_ERR_NONE) {
    Serial.println("SX1262 Initialized!");
  } else {
    Serial.print("Radio failed, code: ");
    Serial.println(state);
    while (true); // Hard halt
  }
}

void loop() {
  // 1. Read the physical sensor (e.g. analog thermistor voltage)
  int raw_temp = analogRead(34);
  
  // 2. Format a simple payload
  String payload = "NODE_01:TEMP:" + String(raw_temp);
  
  // 3. Transmit the packet into the airwaves
  Serial.println("Transmitting...");
  radio.transmit(payload);
  
  // 4. Sleep to prevent battery drain
  delay(15000); 
}
Step 3: The Gateway

Catching the Signal

A second ESP32 node sits on the shore plugged into your laptop via USB. The shore node catches the RF wave, converts it back to digital logic, and pushes it over the USB serial port to the Python Gateway.

gateway/listen.py 
import serial
import time

# Open the physical USB port connected to the Shore Relay
ser = serial.Serial('COM3', 115200, timeout=1)

print("Listening for OMEGA Swarm...")

while True:
    if ser.in_waiting > 0:
        # Read the raw byte buffer from the USB connection
        raw_line = ser.readline()
        
        # Decode the bytes back into a human-readable string
        packet = raw_line.decode('utf-8').strip()
