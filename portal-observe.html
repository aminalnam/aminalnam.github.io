Unified Overlay Standard.

OMEGA-wave doesn't just read data from its own mesh nodes. It contextualizes fleet telemetry against massive global datasets. By actively calling the Global Fishing Watch 4Wings API, Copernicus Marine APIs, and NASA GIBS endpoints, the portal generates live contextual "Workbench" overlays.

The Caching Engine

The FastAPI gateway implements strict Cache-Control headers (`max-age=60`, `max-age=120`) to prevent overwhelming external providers. For example, stormglass_status caches are maintained locally to ensure the portal remains highly responsive even under heavy operator load.

# gateway/api/routes.py (Example Concept)
@app.get("/v1/observe/overlays/gfw")
def get_gfw_fishing_effort(bbox: str):
    # Enforce same-origin proxy to prevent CORS issues on the frontend
    if is_cached("gfw", bbox):
        return cache.get("gfw", bbox)
        
    # Query 4Wings API for gridded effort stats
    raw_data = gfw_client.fetch_bins(bbox)
    normalized = standard_overlay_format(raw_data)
    
    cache.set("gfw", bbox, normalized, max_age=120)
    return normalized
